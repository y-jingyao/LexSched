## 版本信息

- **当前版本**: v0.0.1

## ✨ 功能特性

- 📅 **周视图展示** - 清晰展示一周的课程安排
- 📝 **课程管理** - 添加、编辑、删除课程
- 🎨 **颜色标记** - 不同课程使用不同颜色区分
- 📆 **当前周高亮** - 自动高亮显示当天日期
- ⚙️ **学期设置** - 自定义学期开始日期
- 📥 **数据导入** - 目前仅支持手动
- 🗑️ **缓存清理** - 一键清除本地缓存

## 🚀 技术栈

- **框架**: 微信小程序原生开发
- **样式**: WXSS
- **逻辑**: JavaScript (ES6)
- **配置**: JSON

## 📝 使用说明

1. **查看课程表**: 打开小程序即可看到本周课程安排
2. **添加课程**: 点击右上角 "+" 按钮，填写课程信息
3. **编辑课程**: 点击课程卡片，选择"编辑"
4. **删除课程**: 在课程详情页点击"删除"按钮
5. **设置学期**: 在"设置"页面设置学期开始日期
