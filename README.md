# LexSched 课程表

一款简洁优雅的微信小程序课程表应用。

## 版本信息

- **当前版本**: v0.1.1
- **更新日期**: 2026-03-30

## ✨ 功能特性

- 📅 **周视图展示** - 清晰展示一周的课程安排，支持左右滑动切换周次
- 📝 **课程管理** - 添加、编辑、删除课程，支持自定义上课周次
- 🎨 **颜色标记** - 9种颜色可选，不同课程一目了然
- 📆 **智能日期** - 自动计算当前周次，高亮显示当天日期
- ⚙️ **学期设置** - 自定义学期开始日期
- 🗑️ **数据清理** - 一键清除所有课程数据

## 🚀 技术栈

- **框架**: 微信小程序原生开发
- **样式**: WXSS + CSS 变量（支持深色模式）
- **逻辑**: JavaScript (ES6+)
- **架构**: 模块化设计，分离业务逻辑与UI层

## 🎯 核心功能

### 课程表浏览
- 左右滑动切换周次，带有平滑动画效果
- 上下滑动查看全天课程（1-10节）
- 自动高亮当天日期
- 非本周课程显示灰色标记

### 课程管理
- 点击右上角 "+" 添加课程
- 点击课程卡片查看详情
- 支持设置：课程名称、老师、地点、星期、节次、周次、颜色
- 批量选择上课周次（1-20周）

### 设置选项
- 设置学期开始日期
- 手动添加课程入口
- 清除所有数据（带二次确认）

## 📝 使用说明

1. **首次使用**
   - 打开小程序，自动显示本周课程表
   - 在"设置"页面设置学期开始日期

2. **添加课程**
   - 点击右上角 "+" 按钮
   - 填写课程信息，选择颜色和上课周次
   - 点击"保存"

3. **编辑/删除课程**
   - 点击课程卡片查看详情
   - 点击"编辑"修改信息
   - 点击"删除"移除课程

4. **切换周次**
   - 左右滑动课程表区域切换周次
   - 每次打开自动回到本周
