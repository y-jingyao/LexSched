# 📚 课程表小程序

一个简洁美观的微信小程序课程表应用，帮助学生轻松管理课程安排。
目前仅实现基础功能，后期会逐步完善并开发App版本。
（此小程序并未上线）
## 版本信息

- **当前版本**: v0.0.1
- **更新日期**: 2026-03-29

## ✨ 功能特性

- 📅 **周视图展示** - 清晰展示一周的课程安排
- 📝 **课程管理** - 添加、编辑、删除课程
- 🎨 **颜色标记** - 不同课程使用不同颜色区分
- 📆 **当前周高亮** - 自动高亮显示当天日期
- ⚙️ **学期设置** - 自定义学期开始日期
- 📥 **数据导入** - 目前仅支持手动
- 🗑️ **缓存清理** - 一键清除本地缓存

## 🛠️ 安装运行

1. 克隆项目到本地
2. 使用微信开发者工具打开项目
3. 点击"编译"即可预览

## 📝 使用说明

1. **查看课程表**: 打开小程序即可看到本周课程安排
2. **添加课程**: 点击右上角 "+" 按钮，填写课程信息
3. **编辑课程**: 点击课程卡片，选择"编辑"
4. **删除课程**: 在课程详情页点击"删除"按钮
5. **设置学期**: 在"设置"页面设置学期开始日期
